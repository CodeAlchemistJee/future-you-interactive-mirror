 Future You Interactive Mirror

An interactive desktop application that analyzes your daily habits and predicts your future growth using a visually engaging interface.

 Overview

Future You Interactive Mirror is a Python-based GUI application built with Tkinter that helps users understand how their daily habits impact their long-term success. By adjusting inputs like study hours, screen time, sleep, and consistency, users receive instant feedback on their potential career growth, skill strength, and burnout risk.

Features:
Interactive sliders for:
Study Hours
Screen Time
Sleep Duration

Habit tracking with checkboxes:
Skill Development

Consistency:
Real-time analysis using a custom scoring algorithm

Visual progress bars for:
Career Growth
Skill Strength
Burnout Risk
💬 Personalized motivational feedback
🌙 Modern dark-themed UI
🛠️ Tech Stack
Python
Tkinter (GUI development)
ttk (Themed widgets for modern UI)

🧠 How It Works

The application uses a weighted scoring system:

📘 Study contributes to growth and skill
🔁 Consistency significantly boosts career growth
🧠 Skill development directly impacts skill strength
📱 Excess screen time increases burnout
😴 Proper sleep reduces burnout

Based on these factors, the system:

Calculates scores (0–100 range)
Updates progress bars
Displays a motivational message
